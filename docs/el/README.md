Αυτό είναι ένα καλλιτεχνικό project που δημιουργήθηκε με την πρόθεση να σας διδάξει πώς να δημιουργήσετε ένα pull request.

Επισκεφθείτε το [open-pixel-art.com](https://open-pixel-art.com) για να δείτε το ολοκλήρωμενο έργο!

Οποιοσδήποτε επιθυμεί να μάθει πως να δημιουργήσει ένα  pull request για κάποιο project στο GitHub μπορεί να χρησιμοποιήσει αυτό το project για να συνεισφέρει ένα pixel στον canvas. Θα μπορέσετε να δημιουργήσετε μόνο ένα pixel αλλά μπορείτε να διαλέξετε οποιοδήποτε χρώμα επιθυμείτε με την προϋπόθεση να έχει έναν έγκυρο HEX κωδικό. Για παράδειγμα: `#F22F46` το κύριο χρώμα του [Twilio](https://www.twilio.com).

> Αν είστε αρχάριος στο ανοιχτό λογισμικό, στο GitHub ή στο git γενικά, δεν υπάρχει πρόβλημα, δημιουργήσαμε ένα tutorial [TwilioQuest](https://twil.io/hacktoberfest-quest) για εσάς. Θα σας διδάξει βήμα προς βήμα πως να ξεκινήσετε και θα σας καθοδηγίσει στο πως να δημιουργήσετε ένα pull request.
> 🕹 [Κατεβάστε το TwilioQuest](https://www.twilio.com/quest/download)

Αυτό το project είναι αυτοματοποιημένο και διατηρείται ως επί το πλείστον από bots που θα επαληθεύσουν όλες τις συνεισφορές pixel. Ωστόσο έαν επιθυμείτε να μάθετε περισσότερα για το project ή να υποβάλλετε άλλες συνεισφορές για το project που δεν είναι ένα pixel, μην διστάσετε να ανοίξετε ένα [GitHub issue](https://github.com/twilio-labs/open-pixel-art/issues) μέσα στο [Open Pixel Art project](https://github.com/twilio-labs/open-pixel-art).

## Συνεισφορά

Προκειμένου να συμβάλετε ένα pixel στον canvas, θα πρέπει να δημιουργήσετε ένα  [pull request](https://opensource.guide/how-to-contribute/#opening-a-pull-request) στο [Open Pixel Art project στο GitHub](https://github.com/twilio-labs/open-pixel-art).

Εάν έχετε ήδη εμπειρία με το git και γνωρίζετε πώς να δημιουργήσετε ένα pull request στο GitHub, σας συνιστούμε να ελέγξετε τον [οδηγός συμβολής μας](https://github.com/twilio-labs/open-pixel-art/blob/master/CONTRIBUTING.md).

Αντιλαμβανόμαστε ότι η συμβολή στον ανοικτό κώδικα μπορεί να είναι εκφοβιστική και ως αποτέλεσμα δημιουργήσαμε μια αποστολή στο διαδραστικό παιχνίδι κωδικοποίησης μας [TwilioQuest](https://twil.io/hacktoberfest-quest) που θα σας οδηγήσει βήμα προς βήμα προς τη δημιουργία ενός pull request για αυτό project και θα σας βοηθήσει σε αυτή τη νέα περιπέτεια προς τον ανοικτό κώδικα!

- [Κατεβάστε το TwilioQuest](https://www.twilio.com/quest/download)
- Δείτε τους Οδηγούς Συμβολής:
  - [English](CONTRIBUTING.md)
  - [Español](docs/es/CONTRIBUTING.md)
  - [Deutsch](docs/de/CONTRIBUTING.md)
  - [Brazilian Portuguese](docs/br/CONTRIBUTING.md)
